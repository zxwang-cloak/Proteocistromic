- [ ] Replace placeholder author names in `CITATION.cff`.
- [ ] Replace the placeholder GitHub URL in `CITATION.cff`, `README.md`, and the Nature checklist PDF.

- [ ] Archive the release to Zenodo if a DOI is needed.
